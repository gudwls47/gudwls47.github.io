금융 문서 AsciiDoc 변환 및 LLM 성능 검증

신한투자증권 AI솔루션부 · 2025.09

2025

사내 금융 문서(ISA, 연금저축, 공통업무 매뉴얼 등)를 LLM이 정확히 인식하도록 AsciiDoc 형식으로 변환하고, PDF 대비 응답 품질 향상을 정량적으로 검증했습니다. GPT-5, Gemini 2.5 Flash, Gemini 2.5 Pro 세 모델을 동일 질문으로 비교해 할루시네이션 발생 패턴과 원인을 분석했습니다.

  • ISA·연금저축·공통업무 매뉴얼 등 금융 문서 AsciiDoc 수기 변환 (Mermaid flowchart 포함)
  • PDF 직접 질문 시 LLM이 수수료·표 구조를 오인식하는 케이스 발견 및 원인 분석
  • AsciiDoc 변환 후 동일 질문 재실험 → 인식 정확도 개선 효과 검증
  • Mathpix OCR + Pandoc 자동 변환 워크플로우 구축 (PDF → Markdown → AsciiDoc)
  • LlamaParser · Gemini · 무료 OCR 3가지 도구의 표 구조 복원율 비교 분석
  • GPT-5 vs Gemini 2.5 Flash vs Gemini 2.5 Pro 모델별 오류 유형 및 답변 품질 비교
  • AI PB서비스 출시 후 QA 개선 회의 참여

ADHD를 위한 일상 플래너

루틴 · 투두 · 일기를 하나로 통합한 Android 앱

2026

ADHD 사용자의 실행력을 높이기 위해 루틴·투두·일기를 하나로 통합한 Android 앱입니다. Gemini 2.5 Flash API를 연동해 자연어 또는 사진 입력만으로 루틴과 할 일 목록을 AI가 자동 생성하며, 요일 스케줄도 자동으로 인식합니다.

  • 로그인·홈·마이페이지 화면 개발 및 Retrofit2로 백엔드 REST API 연결
  • Gemini 2.5 Flash 연동 — 자연어·사진 입력 → AI 루틴/리스트 자동 생성
  • AI 요일 스케줄 자동 인식 ("매주 금요일" 입력 시 금요일 스케줄 자동 설정)
  • 월간 잔디 캘린더로 루틴·투두 수행 현황 시각화
  • 일기 작성 시 완료된 루틴·투두 항목 자동 태그 기능 구현
  • 팀장으로서 일정 관리 및 발표 자료(PPT) 제작 주도
Java (Android) Gemini 2.5 Flash Node.js · Express PostgreSQL Retrofit2 Google Sign-In JWT

당뇨 환자를 위한 냉장고 레시피 추천 시스템

CGM 실측 데이터 기반 혈당 상승 예측 + 레시피 추천 ML

2026

냉장고 재료(한국어/영어)를 입력하면 Food.com 레시피 231,637개에서 만들 수 있는 레시피를 추천하고, 45명의 실제 CGM 데이터로 학습한 Gradient Boosting 모델이 예상 혈당 상승값과 당뇨 적합 여부를 판정합니다.

  • PhysioNet CGMacros 45명 실측 데이터(식사 1,524건) + Food.com 231,637개 레시피 전처리
  • 식전 혈당·탄수화물·단백질·지방·식이섬유를 피처로 Gradient Boosting BG rise 예측 모델 구현
  • 알코올·칵테일 레시피 자동 필터링 및 600 kcal 1인분 기준 영양성분 정규화
  • 한국어 재료명 영어 변환 기능 구현으로 냉장고 재료 직접 입력 지원
  • 대한당뇨병학회 기준(식후 2시간 혈당 180 mg/dL 미만)으로 당뇨 적합성 자동 판정
